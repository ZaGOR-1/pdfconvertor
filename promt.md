# Промт для створення конвертера Word документів в PDF

## Основний запит

Створи сучасну десктопну програму на Python для конвертації документів Word (DOC, DOCX) в PDF з графічним інтерфейсом користувача.

## Технічні вимоги

### Основний функціонал
1. **Конвертація документів**: підтримка форматів .doc та .docx
2. **Пакетна обробка**: можливість конвертувати декілька файлів одночасно
3. **Drag & Drop**: інтуїтивне додавання файлів перетягуванням
4. **Збереження**: автоматичне збереження PDF у ту ж папку або вибір власної директорії

### Графічний інтерфейс (GUI)

**Фреймворк**: Використай CustomTkinter (сучасна версія Tkinter) для створення мінімалістичного та привабливого інтерфейсу

**Дизайн вимоги**:
- Темна/світла тема з можливістю перемикання
- Сучасний Material Design або Fluent Design стиль
- Адаптивний інтерфейс з плавними анімаціями
- Індикатор прогресу для кожного файлу та загальний прогрес
- Drag & Drop зона з візуальною індикацією
- Список файлів з можливістю видалення окремих елементів
- Кнопка очищення списку
- Вибір вихідної папки для збереження PDF

**Елементи інтерфейсу**:
- Заголовок програми
- Drag & Drop область з іконкою та підказкою
- Таблиця/список доданих файлів з колонками:
  - Ім'я файлу
  - Розмір
  - Статус (очікує/обробляється/завершено/помилка)
  - Прогрес бар
- Кнопки:
  - "Додати файли" (файловий діалог)
  - "Очистити список"
  - "Вибрати папку збереження"
  - "Конвертувати"
  - Перемикач теми
- Статус бар знизу з загальною інформацією

### Бібліотеки та технології

**Обов'язкові бібліотеки**:
1. **customtkinter** - сучасний GUI фреймворк
2. **python-docx2pdf** або **docx2pdf** - для конвертації DOCX
3. **pypandoc** (альтернатива) - універсальний конвертер документів
4. **Pillow (PIL)** - для роботи з іконками та зображеннями
5. **tkinterdnd2** - для функціоналу Drag & Drop
6. **threading** / **concurrent.futures** - для асинхронної обробки файлів
7. **pathlib** - для роботи з шляхами файлів

**Додаткові можливості**:
- **win32com** (для Windows) - для роботи з .doc файлами через MS Word
- **pyinstaller** - для створення виконуваного .exe файлу

### Функціональні особливості

1. **Багатопотоковість**: 
   - Конвертація не блокує інтерфейс
   - Можливість обробки декількох файлів паралельно
   - Прогрес бар в реальному часі

2. **Обробка помилок**:
   - Валідація форматів файлів
   - Повідомлення про помилки для окремих файлів
   - Логування помилок у файл

3. **Налаштування**:
   - Збереження налаштувань користувача (тема, остання папка збереження)
   - Конфігураційний файл JSON або INI

4. **Оптимізація**:
   - Перевірка наявності PDF перед конвертацією (з опцією перезапису)
   - Оптимізація використання пам'яті для великих файлів
   - Очищення тимчасових файлів
